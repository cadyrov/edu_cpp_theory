Создайте RAII-класс для работы с файлами, который должен:

4. Корректно обрабатывать ошибки
5. Запрещать копирование
6. Разрешать перемещение

### Тесты:
1. Проверка успешного открытия файла
2. Проверка корректного закрытия файла
3. Проверка чтения из файла
4. Проверка записи в файл
5. Проверка обработки несуществующего файла
6. Проверка запрета копирования

9. Проверка работы с исключениями при операциях
10. Проверка отсутствия утечек ресурсов
