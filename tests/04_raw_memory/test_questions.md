## Вопрос 31 (Сложный)

```cpp
class Buffer {
    unsigned char* data_ = nullptr;
    size_t size_ = 0;
    
public:
    void assign(const int* src, size_t count) {
        if (size_ < count) {
            clear();
            data_ = new unsigned char[count * sizeof(int)];
            std::uninitialized_copy_n(src, count, 
                reinterpret_cast<int*>(data_));
            size_ = count;
        }
    }
    
    void clear() {
        if (data_) {
            std::destroy_n(reinterpret_cast<int*>(data_), size_);
            delete[] data_;
            data_ = nullptr;
            size_ = 0;
        }
    }
};
```

**Варианты ответов:**
A) Код правильный
B) Ошибка: нужно использовать `::operator delete[]` вместо `delete[]`
C) Ошибка: неправильный порядок операций
D) Утечка памяти
