Представьте, что вам поручили написать вариабельную шаблонную функцию add, которая возвращает сумму своих аргументов. Чтобы гарантированно избежать проблем со сложением знаковых и беззнаковых чисел, вы решили, что все аргументы должны быть одного типа. - 

Функция уже готова, и вам осталось только реализовать концепт SameAsFirst, который проверяет, что все аргументы шаблона имеют один и тот же тип.

```cpp
template <typename T, typename... Ts>
requires SameAsFirst<T, Ts...>
auto add(T arg, Ts... args) { return (arg + ... + args); } 
```
