# Задания по шаблонам и лямбдам

## 1. Шаблоны функций

1. Напишите шаблон функции `square`, который возвращает квадрат значения.
2. Напишите шаблон функции `min_value`, который возвращает меньшее из двух значений одного типа.
3. Напишите шаблон функции `add`, который принимает два разных типа и возвращает их сумму.
4. Напишите шаблон функции `print_type_size`, который выводит `sizeof(T)` для переданного значения.
5. Напишите шаблон функции `make_default<T>`, который возвращает значение `T{}`.
6. Напишите шаблон функции `swap_values`, который меняет местами два значения одного типа.
7. Напишите шаблон функции `clamp_value`, который ограничивает значение диапазоном `[low, high]`.
8. Напишите шаблон функции `is_equal`, который сравнивает два значения одного типа.
9. Напишите шаблон функции `apply_twice`, который применяет функцию к значению два раза.
10. Напишите шаблон функции `call_and_print`, который вызывает переданную функцию и печатает результат.

## 2. Перегрузка функций и вывод типов в шаблонах

1. Напишите две перегрузки `print`: для `int` и для `double`.
2. Добавьте шаблонную перегрузку `print` для остальных типов.
3. Проверьте, какая перегрузка вызывается для `int`, `double`, `char`, `std::string`.
4. Напишите шаблон `foo(T value)` и проверьте, что `const int` выводится как `int`.
5. Напишите шаблон `foo(T& value)` и проверьте, что `const int` сохраняет `const`.
6. Напишите шаблон `foo(const T& value)` и проверьте вывод `T` для `int* const`.
7. Напишите шаблон `foo(T* ptr)` и проверьте вывод `T` для `int*` и `const int*`.
8. Напишите функцию `max_value(T a, T b)` и проверьте ошибку для `int` и `double`.
9. Исправьте вызов `max_value(1, 2.5)` явным указанием `T`.
10. Напишите набор перегрузок, где обычная функция выигрывает у шаблона при точном совпадении.

## 3. Шаблон класса и переменной

1. Напишите шаблон класса `Box<T>`, который хранит одно значение.
2. Добавьте в `Box<T>` метод `get`, возвращающий значение.
3. Добавьте в `Box<T>` метод `set`, меняющий значение.
4. Создайте `Box<int>` и `Box<std::string>`.
5. Напишите шаблон класса `Pair<T, U>` для двух значений разных типов.
6. Напишите шаблон переменной `pi<T>`.
7. Используйте `pi<float>` и `pi<double>`.
8. Напишите шаблон класса `Counter<T>` со статическим счетчиком объектов.
9. Проверьте, что `Counter<int>` и `Counter<double>` имеют разные счетчики.
10. Напишите класс с конструктором и проверьте CTAD без явного указания `<T>`.

10. Перепишите простую специализацию функции через перегрузку и сравните читаемость.
